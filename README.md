# ⏰ Python Greeting Program

## 📌 Overview
This is a simple Python project that greets the user based on the current system time. It automatically displays **Good Morning**, **Good Afternoon**, **Good Evening**, or **Good Night** according to the current hour.

## ✨ Features
- Displays the current time
- Automatically detects the current hour
- Greets the user based on the time of the day
- Beginner-friendly Python project
- Uses Python's built-in `time` module

## 🛠️ Technologies Used
- Python 3
- Built-in `time` module

## 🚀 How to Run

- Open the project in VS Code.
- Click the **Run** button.
- The program will run automatically.

## 💻 Sample Output

```text
14:30:45
14
30
45
Good Afternoon Sir
```

## 📖 How It Works

The program checks the current system time and prints a greeting based on the current hour.

- 🌅 4 AM – 12 PM → Good Morning
- ☀️ 1 PM – 4 PM → Good Afternoon
- 🌇 5 PM – 7 PM → Good Evening
- 🌙 Remaining Hours → Good Night

- Working with system time
